Font

  • - Color : #000000(default)
  • - point Color : #159B97, #6BD5DD , #FF8585
  • - size (default) : 13px
  • - 'Pretendard' font 사용 : font-family: "Pretendard",sans-serif;
    (기본) NotoSansKR-Regular : font-family:"NotoKR";
    (Light) Pretendard Light : font-weight:300;
    (Medium) Pretendard Medium : font-weight:500;
    (Bold) Pretendard Bold : font-weight:700;
    (Extra Bold) Pretendard Extra Bold : font-weight:800;

Html File Name 규칙

image 네이밍

  • 1. SVG사용, google material icon
  • 2. 자유롭게 네이밍 하되 특정 prefix등을 사용하여 그룹별로 묶일수 있는 네이밍 권장
  • - title: _tit_
  • - background: bg_
  • - icon: ico_
  • - bullet: dot_ , arr_ , bar_
  • - button: btn_
  • - tab: _tab_

class 네이밍 (대시 기법사용)

  • 1. 클래스명은 기본 “대시”표기법으로 사용한다.
  • - input-box, title-box
  • 2. 클래스명명 규칙은 다음과 같다.
  • - “구조명-상태(변화값)” : lnb-link
  • - “css명-변화값” : mt-20
  • 3. 하나에 태그에 여러 개의 클래스 사용할 수 있다.
  • 4. Css에 제공하는 스타일 클래스의 경우 퍼블에서 제공하는 guide 페이지를 참고하여 작업한다.
